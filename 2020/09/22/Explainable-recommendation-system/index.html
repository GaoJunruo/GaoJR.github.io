Explainable recommendation system

要求：
可解释推荐应该从模型和特征两个方面同时具有可解释性
具有可解释的同时，保证正确性
模型的预测和可解释的预测应该是一致的，个性化的

1. Explainable Recommender Systems via Resolving Learning Representations


1).post-hoc methods: some examples include understanding the meaning of latent factors [10, 2). or reconstructing the rationale behind each prediction

缺点：suffers from the issue of explanation accuracy

2).build explainable models or add explainable components, where attention mechanism is commonly applied to highlight important features for prediction


** one of the important building blocks of model explainability is the transparency of representation learning.

2. Explainable Recommendation via Interpretable Feature Mapping and Evaluation of Explainability ∗


提出了个新的评价指标，可以评判可解释性和准确性

以往的方法也提出过可解释的指标：explainability precision (EP) and explainability recall (ER)

本文的指标：从特征角度评估，召回了多少个符合用户偏好的特征。

Top M recall at K (TM@K):评估最喜欢的

Bottom M recall at K (BM@K):评估最不喜欢的

4. Explainable recommendation with fusion of aspect information


a.本文是矩阵分（Aspect-based Matrix Factorization model (AMF)），提出了一种新的评估标准a new metric Satisfaction Degree on Aspects (SDA)
（User Aspect Preference (UAP) and Item Aspect Quality (IAQ)）。

b.“aspect information”：本文认为用户关注item的两个方面the operation aspect 和 the price aspect.
相同的评分，但是可能评论的角度不同，本文回捕获这个方面的差异。

c.用户频繁提及的方面是很重要的方面，一个商品被频繁提到的方面很重要

d.将原始打分矩阵进行分解，用户UAPM和IAQM得到的矩阵限制并重构R。

e.对于推荐结果的评估仍然是传统的方法RMSE和MAE

5.Explanation Mining: Post Hoc Interpretability of Latent Factor Models for Recommendation Systems


本文的解释和表述都很有道理！！重点看！

** as these approaches impose constraints on the black-box model in some way, the model loses flexibility which risks sacrificing some accuracy if the most explainable recommendations are not highly correlated with the most relevant explanations.

** Secondly, even if data can be obtained, if the data sources used for explanations are not correlated with the user-item rating data used to generate recommendations then the explanations will not accurately reflect the reasons for the recommendations, or will not be generated.

** Choosing a simple model which has a logical relationship to the complex model is more likely to yield loyal explanations. We believe there are some natural similarities between matrix factorisation and association rules that could make rules a strong proxy model, particularly for the recommendation problem.

** We hypothesise that these more personalised explanations will have higher fidelity to the black-box recommendation model than the global rules.

本文主要关注：latent factor models 。

contribution：“applying the model-agnostic approach to recommendation systems using matrix factorisation.”

将关联规则和矩阵分解结合，‘矩阵分解和关联规则都是用来建模用户行为的，用关联规则的表达性来描述模型的矩阵分解’

对于可解释性的评估和对于推荐的评估是不同的。对于推荐部分用了precision和recall。对于可解释部分提出一个新的指标。“Model Fidelity”测量解释模型的可信度。

未来工作：
我们有兴趣使用文本数据增强我们的模型，这与用于可解释的协同过滤的主题建模方法类似，这样可以通过理解与某个节目相关的主题，从而使其可解释为另一个节目建议。这一增强将进一步有助于促进其他解释，例如“由于您对主题X感兴趣，我们建议您选择Y”。
